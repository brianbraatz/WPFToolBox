Proportional Window Sizing

Download
File Details
Downloads: 331 File Size: 13.1kB
Posted By: ifpx895 Views: 1273
Date Added: 31 Jan 2007

This sample demonstrates how to proportionally size the window when it is resized by the user by changing either the height of the width using the mouse.  The factor we use in this sample is '1'.

 The core idea to is to listen to WM_SIZING window message and modifty the RECT to ensure that height and width changes in proportion.
