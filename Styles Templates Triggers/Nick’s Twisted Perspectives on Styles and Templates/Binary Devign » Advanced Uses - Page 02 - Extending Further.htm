Advanced Uses - Page 02 - Extending Further

BasedOn

Now that we’ve seen how styles can help one manage the look of their controls in a cleaner fashion we can take things to the next step using the BasedOn property. Using the same glamorous style from earlier, the example below shows how BasedOn can help us to further extend our designs.

Example 04 - Xaml - BasedOn

<Page
   xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
   xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
   xmlns:sys="clr-namespace:System;assembly=mscorlib">
   <Page.Resources>
      <Style x:Key="PinkButton" TargetType="{x:Type Button}">
         <Setter Property="Background" Value="Pink" />
         <Setter Property="Foreground" Value="Purple" />
      </Style>
      <Style x:Key="PinkButtonAdvanced" TargetType="{x:Type Button}" BasedOn="{StaticResource PinkButton}">
         <Setter Property="BorderBrush" Value="Red" />
         <Setter Property="FontSize" Value="20" />
      </Style>
   </Page.Resources>
   <StackPanel Width="300" Height="300">
      <Button Style="{StaticResource PinkButton}">
         Old School Pink
      </Button>
      <Button Style="{StaticResource PinkButtonAdvanced}">
         New School Pink
      </Button>
   </StackPanel>
</Page>

Example 04 - ScreenShot - BasedOn

The BasedOn property allows us to take the properties from the PinkButton Style and apply them to our PinkButtonAdvanced style without having to duplicate the markup. We can then add other settings to our style knowing that if anything changes in the PinkButton style it will be reflected in the PinkButtonAdvanced style.

The only exception to this rule would come about if you were to set a property in the PinkButtonAdvanced Style that was set in the PinkButton Style. Then you would override the inherited value. For example: If the PinkButtonAdvanced Style were to set the Background property to Blue then the pink background would be overridden and a blue background would be drawn.

Just think about it! With the BasedOn property you could extend our virile button to unknown heights of masculinity!
