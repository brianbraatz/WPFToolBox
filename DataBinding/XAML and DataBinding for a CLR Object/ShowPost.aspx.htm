Re: XAML and DataBinding for a CLR Object
Was this post helpful ?

you can create an objectdataprovider

<Window.Resources>

<ObjectDataProvider x:Key="odp1" ObjectType="{x:Type loc:Counter}"></ObjectDataProvider>

</Window.Resources>

and bind the Text of the TextBlock

<TextBlock Name="txt1" Text="{Binding Source={StaticResource odp1}, Path=Value}"></TextBlock>

<Button Name="btn1">up</Button>

</StackPanel>

for example in the code behind when the button is clicked

void btn1_Click(object sender, RoutedEventArgs e)

{

BindingExpression be = BindingOperations.GetBindingExpression(txt1, TextBlock.TextProperty);

Counter obj = be.DataItem as Counter;

obj.CountUp();

}

Your class should implement INotifyPropertyChanged for the Bindings to be updated
