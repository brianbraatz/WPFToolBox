The DWG/DXF to XAML Conversion Toolkit (D2X)

The DWG/DXF to XAML Conversion Toolkit is designed to help developers leverage 2D and 3D DWG and DXF data in Windows Presentation Foundation applications. (See the Samples page for a quick demonstration of the conversion capabilities of the toolkit.)


Supercar model by Mick Imrie and Austin Tate. For more information, see the note at the bottom of this page.

The DWG and DXF formats have been two of the most widely used formats for the storage and exchange of CAD vector data for more than two decades - D2X has been conceived to help make them first class citizens in the designer-developer workflow promise of Windows Presentation Foundation.

So, then what exactly is D2X?

D2X is a development toolkit for .NET developers looking to leverage DWG/DXF data in their applications, and/or to convert it to XAML. It includes everything a developer needs to begin leveraging 2D and 3D DWG and DXF content in their applications, or application workflow. For a more detailed description of D2X's features, check out the features page.

D2X Tooklit Contents

Cryptic or missing documentation, strange usage conventions, and difficult to understand functionality have all been the downfall of some otherwise very useful class libraries. From the very beginning of D2X's development, simplicity, ease of integration, and usability have been key concerns.

The D2X Toolkit includes:
  • The D2XLib .NET Assembly containing classes which can load and convert DWG and DXF files into their WPF/XAML representations
  • D2XLib Documentation detailing the usage of D2X in your applications, dependencies, deployment, and usage of generated files in applications like Expression Interactive Designer
  • The D2X Sample Application which illustrates loading and converting DWG and DXF files using D2X
  • A set of 2D and 3D Sample files for illustration purposes, and for use with the sample application

For some examples of files converted using D2X, check out the Samples page.


The initial version of the D2X Toolkit has been released! Head on over to the releases page, and get it now!


Happy developing,
