WPF Drag and Drop iPod Player

October 06th 2006 Posted to Examples, C# Code, XAML, Video, Sparkle

Today I wanted to learn how to handle drag and drop in WPF so I made this simple media player. I first took a photo of an iPod and modified in Photoshop. I brought that into Sparkle and made the main window transparent. Then in Visual Studio I coded up the C# code to handle the drag and drop and the media controls. Since I'm using the MediaElement control, it can play music, video, and display graphics. You can click anywhere to move it around the desktop. I even included a scrolling text readout that is common on most media players. Click the image to get the app and the source and also check out the C# code below that.
