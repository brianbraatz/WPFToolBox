Arcane Lessons

July 17th, 2007 — arcanecode

Often topics are too big to cover in a single blog post. Thus they get broken up over several days, or sometimes several months. In order to make it easier to review all of the posts for a single topic, I’ve organized them into lessons, below. For each post you’ll see the date of the blog post in MM-DD-YYYY format (Month-Day-Year) followed by the title. I’ve also broken with my usual habit of posting the full URL and instead posted as a hyperlink from the lesson name, since everything posts back to this blog. If you aren’t the trusting type, you can always use the Wordpress calendar control to navigate to the post.

Finally what you see below is a starting point, be sure to check back over the next few weeks as I’ll be adding as I work back in time.
