A Transparent Analog Clock

Here is an analog clock application for Windows Presentation Foundation created with Expression Interactive Designer and Visual C# Express Edition. To use this on WindowsXP you need some stuff from this Get The Beta page.

As far as I know, WinFX windows can't be transparent (at least when using Windows XP), and the Win32 HwndSource host method demonstrated in Unni Ravindranathan's sample Stock Watch Widget doesn't seem to easily allow dragging or resizing (invisible) windows.

I am quickly discovering that things can usually be achieved in any number of ways, and this represents my first solution for getting the best of both worlds. It is an edited version of my recent XAML Analog Clock that creates and disposes of the HwndSources on demand while simultaneously collapsing and restoring the "original" window. The entire clock graphic is captured as a VisualBrush from the original window for use in the transparent window.
