WebcamMicroViewer, a complete solution for streaming images (frame-by-frame) on mobile clients (cellphones and PDAs).

The server it is written entirely in Java using Java Servlet and Java Media Framework technologies, it has all the advantages of the Java platform: security, robustness, portability, scalability.

There are two versions of mobile clients:
- for J2ME-enabled devices (including devices with Palm OS)
- for PocketPCs

The images are delivered according to each client capabilities (image size and bit depth), thus it has less CPU and memory consumption on the client and a faster rate of image refresh.
