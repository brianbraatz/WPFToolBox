Code Tip: Getting an Application's Execution Path

Here's how to get the execution path for a WPF application in .NET 3.5.

string CurrentApplicationPath = System.IO.Path.GetDirectoryName(Application.ResourceAssembly.Location);

Mostly, I'm just blogging it here for future reference and, maybe, someone out there will suggest a better method.
