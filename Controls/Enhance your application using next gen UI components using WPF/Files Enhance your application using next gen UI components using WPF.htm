Using Resource Dictionary

Windows Presentation Foundation is arrived with next generation user experience which makes you always happy to work in applications along with Windows Vista. We have been asked to deliver application interface with look and feel of Windows Vista in our existing applications for the users running Windows XP. In the same direction we have adopted WPF based application development using Blend and Microsoft Visual Studio.

Yeah, looks cool!

We have designed some controls using Simple Styles over WPF basic controls. Every control is associated with individual Resource Dictionary, so no worries if client wants you to change any component’s look or behavior on the fly.  Let me come up with some in depth component details.

Every component brush is defined in a single XAML definition which is used in every component.
