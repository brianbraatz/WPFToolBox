Standard functions in compact package

Get the LED out!

This is a clock gadget made with Expression Interactive Designer and Visual C# Express Edition for use with Windows Presentation Foundation. To use this on WindowsXP you need some stuff from this Get The Beta page.

The clock is actually in a visible window, which forces a rectangular shape and adds a bit of client-system specific chrome (the outer border). Otherwise it is approaching traditional gadget functionality:

  • Draggable by any non-button area
  • Minimal standard Window elements (no Title Bar, Minimize Button, &c.)
  • Controls that are present have custom appearance and behavior
  • Fully resizable taking advantage of vector flexibilty

Options Pane

Select your display preferences.

This is my second clock, and has only the most standard functions. It displays the date on mouse over. Also, you can select 12-hour or 24-hour time display, and MM-DD-YY or DD-MM-YY date display format.

This project is almost completely editable from the Expression Interactive Designer GUI, although I did a lot of code editing along the way. It utilizes styles and templates pretty extensively, among other things, in an attempt to make the appearance easily skinnable.

Shrinky-Dink

The vector based graphics make for clean scaling, even down to small sizes that would become distorted with traditional techniques.
