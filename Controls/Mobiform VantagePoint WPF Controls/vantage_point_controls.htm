VantagePoint WPF Controls

VantagePoint WPF Controls is a suite of user interface components designed for use by software developers and designers using Microsoft .NET 3.0 as the platform for their software products.  VantagePoint consists of Buttons, Tanks, Pipes, Sliders, Radial and Linear Gauges, Motors, and other interface controls that can be used in WPF (Windows Presentation Foundation) Windows or Web Applications.  VantagePoint controls are highly customizable with dozens of properties for configuring the look and feel of the components.  Mix the simplicity of VantagePoint with the power of WPF and you'll find creating intuitive and compelling screens has never been easier!
